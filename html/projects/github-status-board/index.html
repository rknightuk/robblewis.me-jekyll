GitHub Status Widget

A widget for Panic's Status Board for iPad to show GitHub status. Queries status.github.com every 15 minutes to check for changes.

Installation

To manually add the widget, select the "Do-It-Yourself" widget and set the panel URL to

http://robblewis.me/projects/github-status-board/widget/
