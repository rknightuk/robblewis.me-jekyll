Hi. I’m Robb Lewis, a developer living in Portsmouth, UK. I am currently studying Web Technologies at Portsmouth University. I'm not a designer (and don't pretend to be), so I work with designers to create functional sites and applications from their designs.

I do all my development work with an 27″ i3 iMac, a 2011 11″ i5 Macbook Air, an iPad Mini and an iPhone 4S. Most of my writing is done in Byword (iOS and OS X) and MarsEdit with everything synced via Dropbox. I write code in Sublime Text and version it with Git/GitHub. I also use Mike Rogers’ LivePage Chrome extension.

I like trying out new web services and social networks but I generally only use Twitter. If you’re thinking about following me you should read this first.

I would love to start my own podcast but time is not in abundance at the moment. If, however, you would like a guest with a great knowledge of development, technology or the web in general (or Back to the Future), get in contact.
